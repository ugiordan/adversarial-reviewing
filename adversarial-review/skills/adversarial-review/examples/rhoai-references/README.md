# RHOAI Reference Module Examples

These are example reference modules for Red Hat OpenShift AI (RHOAI) strategy reviews. They demonstrate how to write organization-specific reference modules for the adversarial-review tool.

## Usage

To use these in your reviews, copy them into your project's `.adversarial-review/references/` directory or into `profiles/strat/references/<specialist>/`.

See the [reference module expansion plan](../../site/docs/plans/reference-module-expansion.md) for the full reference module system design.

## Files

- `rhoai-auth-patterns.md`: Approved authentication and authorization patterns
- `rhoai-platform-constraints.md`: Platform constraints and deployment assumptions
- `productization-requirements.md`: Red Hat productization checklist (docs, upgrade, telemetry, security, testing)
